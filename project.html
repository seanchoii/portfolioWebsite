Pixel Adventure is a fun little game that I made with graphic images created by Pixel Frog.

I coded this game in Dart language, and used an open source framework by Google called Flutter to run the application on chrome (It can be run on many other platforms). I have also implemented the use of the Flame Game engine that provides a complete set of unique solutions for games.
